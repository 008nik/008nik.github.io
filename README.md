# nikhilshah.de

Personal website of Nikhil Shah - innovation management and foresight researcher at the Applied Innovation Lab, August-Wilhelm-Scheer-Institut, Saarbrücken.

The site is one hand-written HTML file. No framework, no build step, no dependencies - open `index.html` in a browser and that is the whole site. All styling and the small amount of JavaScript (scroll reveals) are inlined.

## Stack

- A single static `index.html`
- Hosted on GitHub Pages at [nikhilshah.de](https://nikhilshah.de)
- Light and dark theme via CSS custom properties and `prefers-color-scheme`
- No cookies, no tracking, no external requests

## Local preview

Open `index.html` in any browser. Nothing to install.
